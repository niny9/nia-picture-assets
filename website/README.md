# Personal Website Image Assets

Store images for the personal website in this directory.

Recommended naming format:

- `profile-main.jpg`
- `travel-diving.jpg`
- `ski-selfie.jpg`
- `horse-riding.jpg`
- `climbing-wall.jpg`
- `running-certificate.jpg`

CDN base:

`https://cdn.jsdelivr.net/gh/niny9/nia-picture-assets@main/website/`

Example HTML:

```html
<img src="https://cdn.jsdelivr.net/gh/niny9/nia-picture-assets@main/website/profile-main.jpg" alt="Profile photo">
```
